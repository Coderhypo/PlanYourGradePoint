教务处信息来源

所有的数据来源自教务处的“山东理工大学学生成绩查询系统”，如果这系统没挂的话应该保证数据的准确性。

预计重修后绩点

绩点不够又不想放弃学位证书？打算重修不知道靠不靠谱？那么就预算下重修得到的成绩对绩点的影响吧。

使用姿势

输入学号之后计算当前绩点，修改后面的成绩点击更新可以计算成绩修改后的绩点。如果遇到 bug 可以联系作者，或者去 Github 提 pr。
